### 책을 사랑하는 사람들을 위한 서비스

#### 독서 기록을 앱으로 편하게 남겨보세요!

: 읽은 책을 기록하고 독후감/추천/문장 수집 등의 게시물을 업로드할 수 있습니다.

<br>

### 프로젝트 설명

이 프로젝트는 개인 프로젝트입니다. 

- **Backend**: Java 17, Spring Boot 3.5.7, MySQL  
- **Frontend**: Swift
- **Deploy**: AWS EC2, RDS

<br>

---

<br>

### 주요 기능

#### 📖 1. 다 읽은 책 / 읽고 있는 책 / 읽다 만 책 / 읽고 싶은 책을 서재에 추가

- 서재에 책을 분류해서 추가할 수 있어요. 

<img width="7732" height="4546" alt="addbook" src="https://github.com/user-attachments/assets/3ef7d7dd-596e-493a-8900-e716c6f39f7a" />



#### 📖 2. 서재에서 책을 상태별로 분류해서 조회

- 완독한 책은 별점 별로 모아볼 수 있고, 그 외에도 읽고 있는 책이나 읽다 만 책, 읽고 싶은 책을 조회할 수 있어요. 책을 클릭하면 책의 상세정보를 조회하거나 상태를 변경할 수 있습니다. 

<img width="7732" height="4546" alt="library" src="https://github.com/user-attachments/assets/48b430fb-5f84-411b-b041-876b2f4ae63c" />
<img width="7732" height="4546" alt="bookdetail" src="https://github.com/user-attachments/assets/4e576a76-1e45-4b3d-b908-d1554988d245" />



#### 📖 3. 독후감 / 추천&비추천 / 문장 수집 게시물 작성

- 책에 대한 기록을 남길 수 있어요. 공개/비공개 전환이 가능하며 공개로 업로드한 게시글은 모두가 볼 수 있습니다. 댓글이나 대댓글을 달 수도 있어요.
- 게시글을 보면서 마음에 드는 책은 바로 위시리스트에 추가할 수 있어요.
